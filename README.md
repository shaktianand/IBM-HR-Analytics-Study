# IBM-HR-Analytics-Study
End‑to‑end HR analytics workflow applied to the IBM HR dataset, highlighting attrition drivers and workforce trends. The project features interactive Tableau dashboards that visualize various factor driving attrition, supported by Excel data preparation.
